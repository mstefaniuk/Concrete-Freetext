Content assisted free text editor for Concrete metamodel

Concrete editor is powerful web-based DSL editor. It uses structured approach where appearance can be highly customized with HTML/CSS templates. Creation of a language is based on defining specific metamodel and optionally javascript plugins.

It is structured editor so new entities are created with set of attribute placeholders. It is convenient in most situations but in some cases not. Expression languages (arithemtic or XPath) are easier to edit in free text manner. Certain level of precision and convenience in that case can be provided by content completion.

I implemented simple free-text editor for Concrete metamodel. Main idea was to create language looks similar to natural language. It is not polished too much but it shows the concept.
